📖 Google Sheets 연동 가이드

아래 Apps Script 코드를 Google Sheets에 붙여넣고 배포하면 연동이 완료됩니다.

  1. Google Sheets 열기 → 상단 메뉴 확장 프로그램 → Apps Script
  2. 아래 코드 전체를 붙여넣고 저장 (💾)
  3. 배포 → 새 배포 → 유형: 웹 앱
    실행 계정: 나 / 액세스 권한: 모든 사용자
  4. 생성된 웹 앱 URL을 설정 탭에 입력
